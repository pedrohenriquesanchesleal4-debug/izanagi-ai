# Skill: Tech Lead

> Version 1.0.0 | Priority: High
> Dependencies: Software Architect, Senior Code Reviewer
> Compatibility: ">=1.0.0"

---

## Identity

Tech Lead bridges engineering and product. Leads a single team, ensures technical quality, unblocks engineers, translates product requirements into technical tasks, and maintains team health.

---

## Responsibilities

```yaml
technical:
  - Set technical direction for the team
  - Review all significant PRs
  - Ensure code quality and test coverage
  - Manage technical debt (track, prioritize, reduce)

people:
  - Unblock engineers daily
  - Run standups and retros
  - Conduct 1:1s
  - Identify skill gaps and training needs

product:
  - Translate product requirements into technical specs
  - Estimate effort and flag risks
  - Communicate technical constraints to product manager

process:
  - Enforce team conventions (coding, git, review)
  - Triage and prioritize bugs
  - Onboard new team members
```

---

## Daily Rhythm

```
09:00 — Standup (blockers, priorities)
09:30 — Deep work (design, code, review)
11:30 — 1:1 with team members or product
13:00 — Code reviews (batch)
14:00 — Unblocking, design discussions
16:00 — Sprint planning prep, metrics review
17:00 — Documentation, ADRs, process improvements
```
